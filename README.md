# Настройки и запуск проекта

## 1. Настройки базы данных и приложения

Проект использует **Spring Boot** и **PostgreSQL**. Перед запуском приложения необходимо подготовить базу данных и настроить подключение.

### 1.1. Создание базы данных

1. Убедитесь, что PostgreSQL установлен и запущен.
2. Создайте базу данных с помощью командной строки или pgAdmin

### 1.2. Настройка подключения
В проекте используется файл src/main/resources/application.properties. Необходимо указать следующие настройки:

# Подключение к базе данных PostgreSQL
spring.datasource.url=jdbc:postgresql://localhost:5432/<Ваше название бд>
spring.datasource.username=<Ваш пользователь>
spring.datasource.password=<Пароль пользователя>

### 1.3. Проверка подключения
После настройки запустите проект. Если подключение установлено правильно, Spring Boot автоматически создаст необходимые таблицы в базе данных.

### 2. Сборка проекта
Проект собирается с помощью Maven.
Перейдите в корень проекта, где находится pom.xml. Выполните команду:
mvn clean package

После сборки будет создан JAR-файл в директории target/:
target/library.jar

### 3. Запуск приложения
Запустите приложение с помощью java -jar из корня проекта:
java -jar target/library.jar

Приложение будет доступно по адресу:
http://localhost:8081

Перейдите на одну из страниц приложения, например:
http://localhost:8081/borrows
